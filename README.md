# ⚖️ مساعد الردود للمستفيدين
### نظام المرافعات الشرعية

مساعد ذكي يساعد موظفي المحاكم على صياغة الردود الرسمية للمستفيدين، مع ربط كل رد بالمواد النظامية ذات الصلة من نظام المرافعات الشرعية السعودي.

🔗 **[العرض التجريبي المباشر](https://abosalehg-ui.github.io/legal-assistant/)**

---

## 📌 الفكرة

يواجه موظفو المحاكم تحديًا يوميًا في صياغة ردود رسمية دقيقة وسريعة على استفسارات المستفيدين، مع الحرص على توافق الرد مع النظام. هذا المشروع يحل هذه المشكلة بتوفير أداة تجمع بين تحليل الرسائل الواردة، واقتراح المواد النظامية المناسبة، وتحسين صياغة الرد.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 🔍 **تحليل الرسائل** | تحليل تلقائي لرسالة المستفيد وتحديد موضوعها القانوني |
| 📚 **المواد النظامية** | عرض المواد ذات الصلة من نظام المرافعات الشرعية تلقائيًا |
| ✍️ **كتابة الرد** | واجهة لكتابة الرد مع دعم اللهجة العامية |
| ✨ **تحسين الصياغة** | تحويل الرد إلى صياغة رسمية احترافية |
| 📎 **إضافة مواد** | إمكانية إضافة مواد نظامية إضافية للرد |
| 💾 **حفظ الردود** | حفظ الردود النهائية لاسترجاعها لاحقًا |
| 📋 **قوالب جاهزة** | مجموعة من القوالب الجاهزة للحالات الشائعة |

---

## 🖥️ واجهة المستخدم

التطبيق مقسم إلى أربع مناطق رئيسية تعمل بتسلسل منطقي:

```
┌─────────────────────────────────────────┐
│  📝 رسالة المستفيد  →  🔍 تحليل       │
├─────────────────────────────────────────┤
│  📚 المواد النظامية ذات الصلة          │
├─────────────────────────────────────────┤
│  ✍️ كتابة الرد  →  ✨ تحسين الصياغة  │
├─────────────────────────────────────────┤
│  📤 الرد النهائي  →  💾 حفظ / نسخ     │
└─────────────────────────────────────────┘
```

---

## 🚀 طريقة الاستخدام

1. **الصق رسالة المستفيد** في حقل النص المخصص
2. **اضغط "تحليل الرسالة"** لاستخراج الموضوع وعرض المواد النظامية ذات الصلة
3. **راجع المواد المقترحة** وأضف أو احذف حسب الحاجة
4. **اكتب ردك** بأسلوبك الطبيعي (حتى بالعامية)
5. **اضغط "تحسين الصياغة"** للحصول على رد رسمي جاهز
6. **انسخ أو احفظ** الرد النهائي

## 🛠️ التقنيات المستخدمة

- **HTML5 / CSS3** — بناء الواجهة
- **JavaScript (Vanilla)** — منطق التطبيق
- **Claude API (Anthropic)** — تحليل الرسائل وتحسين الصياغة
- **GitHub Pages** — الاستضافة المجانية
