# مساعد الردود للمستفيدين

<div dir="rtl">

تطبيق ويب يساعد موظفي خدمة المستفيدين على صياغة ردود رسمية ومهنية، مربوط باللوائح التنفيذية لنظام المرافعات الشرعية في المملكة العربية السعودية.

## 📖 نظرة عامة

يواجه موظفو الرد على استفسارات الجمهور تحديات متكررة في صياغة ردود تجمع بين الدقة النظامية والوضوح اللغوي. يقدّم هذا التطبيق أدوات عملية تُقلّل زمن صياغة الرد وتُحسّن جودته من خلال تحليل رسالة المستفيد واقتراح المواد النظامية ذات الصلة وتحويل الصياغة من العامية إلى الفصحى الرسمية.

## ✨ المزايا الرئيسية

### 🧠 تحليل ذكي لرسائل المستفيدين
- كشف **نبرة الرسالة** (شكوى / استفسار / طلب / عاجل)
- تحديد **الموضوع الرئيسي** من بين 9 أنماط معرَّفة (موعد جلسة، اعتراض إغلاق، متابعة مذكرة، أحوال شخصية، وغيرها)
- استخراج تلقائي للـ **كيانات**: أرقام المذكرات، التواريخ، أرقام الجوال، أرقام الهوية
- كشف **الكلمات المفتاحية** مع دعم لأكثر من 40 مجموعة مرادفات

### 📚 ربط بالمواد النظامية
- قاعدة بيانات تضم مواد من اللوائح التنفيذية لنظام المرافعات الشرعية
- خوارزمية ترجيح متقدمة تُرتّب المواد حسب صلتها بالرسالة
- بحث وتصفية حسب الفئة (التبليغ، الجلسات، الأحوال الشخصية، التنفيذ، إلخ)
- إمكانية إدراج نص المادة مباشرةً في الرد

### ✍️ تحسين الصياغة اللغوية
- قاموس موسّع يضم أكثر من 230 مدخلاً للتحويل من العامية إلى الفصحى
- تصنيف المفردات إلى: أفعال حركة، أفعال رؤية، أدوات استفهام، كلمات قضائية، وغيرها
- دعم البادئات العربية (و، ف، ب، ل، ك) للتعامل مع كلمات مثل "وبتلاقي" و"فيروح"
- تطبيع النص (Normalization) لتوحيد الهمزات والتاء المربوطة وإزالة التشكيل
- قاموس مصطلحات قانونية يحوّل المفردات الدارجة إلى نظيراتها النظامية

### 📋 قوالب جاهزة
ثمانية قوالب مُعدّة مسبقاً للحالات الشائعة: موعد جلسة، تأجيل جلسة، استكمال مستندات، إغلاق طلب، رد على اعتراض، توضيح إجراء، إحالة لجهة أخرى، تبليغ حكم.

### 📤 صندوق رد قابل للتعديل
الرد المقترح قابل للتعديل المباشر قبل النسخ أو الحفظ، بحيث يستطيع الموظف:
- إضافة نصه الخاص
- تعديل القوالب لملء الحقول مثل `[التاريخ]` و `[رقم الطلب]`
- دمج عدة مواد مع تعديلات مخصصة

### 💾 أرشفة الردود
حفظ الردود محلياً واستدعاؤها لاحقاً، مع معاينة سريعة وتاريخ الحفظ.

## 🎯 الاستخدام

1. **الصق رسالة المستفيد** في الصندوق الأول واضغط "تحليل الرسالة"
2. سيعرض التطبيق:
   - ملخص التحليل (نبرة، موضوع، كيانات مستخرجة)
   - قائمة المواد النظامية ذات الصلة مرتبة حسب قوة التطابق
   - مسودة رد أولي قابلة للتعديل
3. **اختياري**: اختر مادة أو أكثر واضغط "إضافة مادة" لإدراج نصها في الرد
4. **اختياري**: اكتب ردك بالعامية واضغط "تحسين الصياغة" لتحويله إلى لغة رسمية
5. **عدّل الرد** في الصندوق النهائي حسب الحاجة
6. اضغط **نسخ** لاستخدام الرد في CRM، أو **حفظ** للرجوع إليه لاحقاً

### المتطلبات
- متصفح حديث يدعم ES6+ (Chrome 60+، Firefox 55+، Safari 11+، Edge 79+)
- اتصال بالإنترنت للمرة الأولى (لتحميل الخطوط من Google Fonts)

## 🛠️ البنية التقنية

- **HTML5** + **CSS3** + **Vanilla JavaScript** (بدون أطر عمل)
- **LocalStorage** لحفظ الردود محلياً
- **Google Fonts**: Tajawal (للنص العام) + Amiri (للرد النهائي)
- **تصميم متجاوب** (Responsive) يدعم الهواتف والأجهزة اللوحية
- **RTL كامل** لعرض مثالي للمحتوى العربي
- **طباعة نظيفة** عبر Print Styles

## 🔧 التخصيص

### إضافة مادة نظامية جديدة

ابحث عن `const legalArticles` في الكود وأضف كائناً جديداً:

```javascript
{
    id: "1/100",
    number: "المادة 1/100",
    title: "عنوان المادة",
    text: "نص المادة كاملاً...",
    category: "الفئة",
    keywords: ["كلمة1", "كلمة2", "كلمة3"]
}
```

### إضافة تحويل عامية جديد

ابحث عن `colloquialToFormal` وأضف مدخلاً:

```javascript
"الكلمة_العامية": "المقابل_الفصيح",
```

### إضافة قالب جديد

ابحث عن `responseTemplates` وأضف:

```javascript
{
    id: 9,
    name: "اسم القالب",
    icon: "📌",
    text: "نص القالب مع [حقول قابلة للتعديل]"
}
```

## ⚠️ تنبيه قانوني

هذا التطبيق أداة **مساعِدة** تقترح مواد نظامية وصياغات بناءً على تحليل نصي. لا يُعدّ التطبيق مرجعاً قانونياً بديلاً عن المصادر الرسمية. المسؤولية النهائية عن صحة الرد ومطابقته للنظام الساري تقع على عاتق الموظف المُستخدِم. يُوصى دائماً بالرجوع إلى أحدث نسخة من اللوائح التنفيذية الصادرة من الجهة المختصة.
