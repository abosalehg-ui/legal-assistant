## 📌 نظرة عامة

تطبيق ويب تفاعلي مصمم لموظفي المحاكم في المملكة العربية السعودية، يُبسّط عملية الرد على استفسارات المستفيدين من خلال ثلاث مراحل متسلسلة: **تحليل الرسالة الواردة** — **استخراج المواد النظامية ذات الصلة** — **صياغة رد رسمي احترافي**.

---

## ✨ المميزات

| الميزة | الوصف |
| --- | --- |
| 🔍 **تحليل تلقائي** | تحليل رسالة المستفيد وتحديد موضوعها القانوني فوراً |
| 📚 **مواد نظامية** | استخراج المواد ذات الصلة من نظام المرافعات الشرعية تلقائياً |
| ✍️ **كتابة مرنة** | كتابة الرد بأسلوبك الطبيعي بما فيه العامية |
| ✨ **تحسين الصياغة** | تحويل الرد إلى لغة رسمية احترافية بضغطة واحدة |
| 📎 **إضافة مواد** | إمكانية إضافة مواد نظامية إضافية يدوياً |
| 📋 **قوالب جاهزة** | مجموعة قوالب للحالات الشائعة توفر الوقت |
| 💾 **حفظ الردود** | حفظ الردود النهائية محلياً لاسترجاعها لاحقاً |
| 📱 **تصميم متجاوب** | يعمل بسلاسة على الهاتف والحاسب |

---

## 🖥️ آلية العمل

التطبيق يسير وفق تدفق منطقي في أربع مناطق:

```
1. 📝 رسالة المستفيد    ──▶  الصق الرسالة الواردة
           │
           ▼
2. 🔍 تحليل + مواد      ──▶  استخراج الموضوع + المواد النظامية تلقائياً
           │
           ▼
3. ✍️ كتابة الرد        ──▶  اكتب بأسلوبك ← ثم حسّن الصياغة
           │
           ▼
4. 📤 الرد النهائي      ──▶  نسخ / حفظ / أرشفة
```

## 🚀 التشغيل

### المعاينة المباشرة

```
https://abosalehg-ui.github.io/legal-assistant/
```

### التشغيل المحلي

```bash
# استنساخ المستودع
git clone https://github.com/abosalehg-ui/legal-assistant.git

# الانتقال للمجلد
cd legal-assistant

# فتح الملف في المتصفح
open index.html        # macOS / Linux
start index.html       # Windows
```

> التطبيق يعمل بالكامل في المتصفح ولا يحتاج خادم أو تثبيت.
